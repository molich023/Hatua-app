# Contributing to HATUA 🇰🇪

We love your help! To keep HATUA secure and efficient:

1. **Bug Reports:** Open an issue with "BUG:" in the title.
2. **Security:** If you find a GPS spoofing vulnerability, email molich60@gmail.com instead of opening an issue.
3. **Localization:** Want to add more languages (e.g., Sheng or Luganda)? Update `src/lib/translations.ts`.
4. **Pull Requests:** - Ensure `npm run lint` passes.
   - Use descriptive commit messages (e.g., `feat: add M-Pesa callback handler`).
